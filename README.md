## 🧩 Descrizione

Questo repository raccoglie esercizi e progetti sviluppati durante il mio percorso di studio. Ogni progetto è costruito utilizzando principi cardine dell’OOP, tra cui:

- **Incapsulamento**
- **Ereditarietà**
- **Polimorfismo**
- **Astrazione**
